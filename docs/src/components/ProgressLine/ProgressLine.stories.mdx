# ProgressLine

> `ProgressLine` is be used to display the bar graph.

<br />

## Usage

```tsx
import {ProgressLine} from 'dooboo-ui';

...
<ProgressLine value={25} />;
```

| Props  | Types             | Description                                                                                                         | Required | Default   |
| ------ | ----------------- | ------------------------------------------------------------------------------------------------------------------- | -------- | --------- |
| styles | ProgressbarStyles | This props determine styles of the component. You don't need to fill this props because the default values are set. |          | undefined |
| value  | number            | Progress value. It must be between 0 to 1.                                                                          | v        |           |
| type   | ProgressbarType   | Progress theme type. This value determined the color of the shape if prop color is undefined.                       |          | "info"    |
